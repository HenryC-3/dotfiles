## 映射

### 规则一

| 原键位        | 映射           | 备注 |
| ------------- | -------------- | ---- |
| caps_lock     | ctrl+alt+cmd   |      |
| right_control | ctrl+shift+cmd |      |
| right_shift   | shift+alt+cmd  |      |
| right_alt     | ctrl+shift+alt |      |

### 规则二

| 原键位 | 映射        | 备注 |
| ------ | ----------- | ---- |
| ctrl+h | left-arrow  |      |
| ctrl+j | down-arrow  |      |
| ctrl+k | up-arrow    |      |
| ctrl+l | right-arrow |      |
